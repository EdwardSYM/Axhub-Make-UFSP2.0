# 专项领域通用工作台 - 规格文档

## 1. 页面定位

`/prototypes/topic-workbench2` 是唯一的“专项领域通用工作台”主入口。

无查询参数直接访问时，默认按 `category=special&topic=key_area_rectify` 展示专项领域通用工作台；从首页“专项监督”进入年度重点检查、审计问题整改、巡视问题整改、监管局问题整改等主题时，也统一进入本页面。

## 2. 适用范围

- `category=special` 下所有主题共用同一套评级体系结构。
- 页面结构以当前“地方政府债务”工作台为准，包含评分、趋势、一级指标、二级指标、三级指标页、规则定义和政策依据。
- 不再只对 `yearly/local-debt` 启用该结构；`yearly/*`、`audit`、`inspect`、`supervision` 等专项主题只切换数据、文案、指标名称、政策依据和分值。
- 日常监督不复用本目录作为主入口，业务监控和专题监控分别进入独立目录。

## 3. 页面结构

- 顶部：面包屑、统计口径、统计期/趋势范围、查看趋势/返回评分、设置入口。
- 左侧评分态：指标下钻或同级指标切换，上方为当前指标组，下方为本期规则得分与权重。
- 左侧趋势态：指标下钻或指标切换，以及本期得分和较上期变化，不在左侧重复展示完整趋势图。
- 右侧评分态：分析结论、规则定义、政策依据。
- 右侧趋势态：趋势曲线、均值/最高/最低/中位数/波动判断和智能分析。
- 第三级页面：延续同一视觉框架，支持指标切换、规则定义、政策依据与趋势查看。

## 4. 数据与交互

- `topic` 决定主题名称、指标树、政策依据、规则口径和示例得分。
- `category=special` 决定使用专项领域评级体系视图。
- 趋势范围由顶部起止周期选择控制，图表与统计卡片同步使用该范围。
- 功能列表页返回工作台时按 `category/topic` 判断回到本页面或日常监督独立工作台。

## 5. 独立维护约束

- 本目录只引用本目录内的 `./data`、`./style.css`、`./LocalDebtLevel3Panel`，以及公共组件和主题。
- 日常监督业务监控与专题监控为物理复制出的独立页面，不从本目录 import 或 re-export。
- 后续专项主题如果补充真实业务数据，只补充本目录数据与文案，不新增平行旧版工作台。
